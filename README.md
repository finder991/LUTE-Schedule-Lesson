# LUTE-Schedule-Lesson
Бот для телеграму з розкладом пар 

# Вимоги

- Python 3.7 або вище
- Бібліотека `python-telegram-bot`
- Токен бота (отримати в [@BotFather](https://t.me/BotFather))

# Встановлення та запуск

1. **Клонуйте репозиторій** (або створіть файл `bot.py` і скопіюйте код з репозиторію)

2. **Встановіть Python 3** (якщо ще не встановлено)  
   [Завантажити Python](https://python.org)

3. **Встановіть залежності**  
   ```bash
   pip install python-telegram-bot --upgrade
   ```
4. **Додайте токен бота**
   У файлі bot.py замініть 'YOUR_TOKEN' на справжній токен:
   ```python
   TOKEN = 'YOUR_TOKEN_HERE'
   ```
5. Запустіть бота
   ```bash
   python bot.py
   ```
